Automatische Bildnachweise

Dieses Bookmarklet erzeugt Bildnachweise nach Spektrum-Schema für picture alliance, Getty Images und Adobe Stock und kopiert das Ergebnis direkt in die Zwischenablage.

Ziehe diesen Button in deine Lesezeichenleiste:

Bildnachweis

Alternativ: Rechtsklick auf den Button → Link als Lesezeichen speichern

So funktioniert’s

  1. Blende die Lesezeichenleiste deines Browsers ein.
  2. Ziehe den Button „Bildnachweis“ per Drag-and-drop in die Leiste.
  3. Öffne anschließend eine Bildseite bei picture alliance, Getty Images oder Adobe Stock.
  4. Klicke auf das gespeicherte Lesezeichen.
  5. Der formatierte Bildnachweis wird direkt in die Zwischenablage kopiert.

Ausgabeformate

  • picture alliance: Fotograf / Agentur / picture alliance
  • Getty Images: Fotograf / Getty Images / Kollektion
  • Adobe Stock: Fotograf / stock.adobe.com
Falls der Browser das direkte Kopieren blockiert, zeigt das Bookmarklet stattdessen ein Feld an, aus dem der Text manuell kopiert werden kann.
